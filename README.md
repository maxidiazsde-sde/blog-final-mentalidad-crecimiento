# Cómo aprendí Normalización de Datos en Data Analytics

## Contexto

Durante la materia Data Analytics tuve que aprender normalización de bases de datos.

## Problema

Al principio me costaba entender las claves primarias y foráneas.

## ¿Qué hice?

Busqué ejemplos.
Practiqué ejercicios.
Consulté al profesor.

## ¿Qué aprendí?

Comprendí cómo relacionar tablas correctamente.

## Reflexión

Aprendí que equivocarse forma parte del proceso de aprendizaje.
